## 一、文档目标

本文档用于把“武将即局内主角”的方案落到可执行层，解决以下问题：

- 首发武将选谁
- 每个武将的局内手感如何区分
- 局外养成如何映射到局内
- 武将如何同时作用于“搜、打、撤”
- 首版应该做到什么程度，避免开发过重

本文档聚焦首发 4 名武将：

- 赵云
- 关羽
- 黄忠
- 诸葛亮

## 二、武将系统设计原则

### 2.1 核心原则

**局外养成的武将，就是局内实际操控的主角。**

即：

- 局外养赵云
- 进局就是赵云
- 赵云的普攻、技能、撤离能力和成长感都必须能被玩家直接感知

### 2.2 首版角色差异来源

首版不通过“大量复杂连招”做差异，而通过以下 5 个层面区分角色：

1. 武器类型不同
2. 普攻节奏不同
3. 主动技能不同
4. 被动与觉醒方向不同
5. 搜打撤专属能力不同

### 2.3 角色成本控制

首版采用“职业模板复用 + 个性技能差异化”策略。

首发 4 人分别覆盖 4 类体验：

| 武将 | 职责 | 玩家感知关键词 |
|---|---|---|
| 赵云 | 先锋 | 灵活、突围、能打能撤 |
| 关羽 | 猛将 | 爆发、重斩、打精英 |
| 黄忠 | 射手 | 稳定、安全、远程压制 |
| 诸葛亮 | 奇谋 | 控场、收益、策略感 |

## 四、首发武将详细设计

## 4.1 赵云

### 4.1.1 基础信息

| 项目 | 内容 |
|---|---|
| 名称 | 赵云 |
| 阵营 | 蜀 |
| 职业 | 先锋枪将 |
| 武器 | 长枪 |
| 获取方式 | 首发默认可得 / 新手引导武将 |

### 4.1.2 核心定位

赵云是首发最适合新手的“通用型主角”。

他在搜打撤中的职责是：

- 能快速切入敌群
- 能快速脱离危险区域
- 能在撤离阶段保持稳定生存

一句话概括：

**赵云是最适合“又想打、又想贪、还想顺利撤”的武将。**

### 4.1.3 普攻模组

设计方向：轻快、连续、灵活。

| 项目 | 设计 |
|---|---|
| 攻击段数 | 3 段 |
| 攻击范围 | 中距离 |
| 节奏 | 快 |
| 动作感 | 刺、挑、穿 |
| 目标手感 | 不笨重，强调连贯与位移衔接 |

建议表现：

- 第一段：直刺
- 第二段：横扫
- 第三段：突刺收尾

普攻手感目标：

- 比关羽快
- 比黄忠更贴脸
- 比诸葛亮更依赖位移和节奏

### 4.1.4 主动技能

#### 固有主动技：七进七出

效果定义：

- 朝当前朝向或目标方向快速突进一段距离
- 路径上的敌人受到穿刺伤害
- 突进期间短暂无敌或高减伤

建议参数方向：

- 冷却：8 秒
- 位移长度：中
- 命中数：可穿透 3-5 个普通敌人

设计意图：

- 作为赵云最强识别点
- 兼顾进攻与撤离
- 让玩家在“危险时刻”有英雄感

### 4.1.5 被动技能

#### 固有被动：龙胆

效果方向：

- 使用位移技能后，短时间内提升暴击率或攻速

设计目的：

- 让赵云形成“位移接输出”的战斗节奏
- 鼓励玩家主动进出战场，而不是站桩

#### 觉醒被动：孤骑断后

效果方向：

- 生命低于一定比例时，获得一次短时间减伤与移速提升
- 每局触发次数有限

设计目的：

- 强化赵云“残局反打、强行突围”的角色认知

### 4.1.6 搜打撤专属能力

| 维度 | 设计 |
|---|---|
| 搜 | 搜索期间若受攻击，获得短暂移速提升，方便脱离 |
| 打 | 位移后短时间普攻强化 |
| 撤 | 撤离读条期间可获得额外减伤或一次免费突进 |

### 4.1.7 局内强化方向

赵云推荐 3 条强化流派：

#### 1. 突进连击流

- 突进次数增加
- 突进后攻速提升
- 普攻末段附带穿透

#### 2. 反杀生存流

- 低血增伤
- 位移后护盾
- 命中回血

#### 3. 撤离保命流

- 撤离读条减伤
- 被追击时移速提升
- 撤离点附近技能冷却缩短

### 4.1.8 局外养成映射

| 养成项 | 局内体现 |
|---|---|
| 等级 | 基础攻击、生命、移动底盘提升 |
| 技能等级 | 七进七出伤害、位移长度、冷却优化 |
| 星级 | 解锁突进追加效果 |
| 觉醒 | 解锁残局保命能力 |
| 专属强化 | 强化撤离与突围能力 |

### 4.1.9 首版实现优先级

赵云必须作为 **P0** 武将实现，原因：

- 最适合做新手默认角色
- 兼容搜、打、撤三段体验
- 能帮助玩家更顺畅理解撤离玩法

## 4.2 关羽

### 4.2.1 基础信息

| 项目 | 内容 |
|---|---|
| 名称 | 关羽 |
| 阵营 | 蜀 |
| 职业 | 猛将刀将 |
| 武器 | 青龙刀 |
| 获取方式 | 首批解锁武将 |

### 4.2.2 核心定位

关羽是“高爆发重斩型主角”。

他在搜打撤中的职责是：

- 快速处理精英敌人
- 通过强斩杀能力缩短高风险战斗
- 让玩家感受到纯粹的攻击爽感

一句话概括：

**关羽不是最稳的武将，但他是最能打出“一刀定胜负”体验的武将。**

### 4.2.3 普攻模组

设计方向：沉重、强力、清晰。

| 项目 | 设计 |
|---|---|
| 攻击段数 | 2 段或 3 段 |
| 攻击范围 | 中近距离扇形 |
| 节奏 | 慢 |
| 动作感 | 横斩、下劈、蓄势重斩 |
| 目标手感 | 每一下都很重，有明显停顿和压制感 |

建议表现：

- 第一段：横斩
- 第二段：上挑或下劈
- 终段：重斩

### 4.2.4 主动技能

#### 固有主动技：青龙断岳

效果定义：

- 短暂蓄力后向前方打出高伤斩击
- 对精英敌人额外造成破甲或斩杀系伤害

建议参数方向：

- 冷却：10 秒
- 范围：前方中距离扇形
- 特点：命中精英时反馈极强

设计意图：

- 建立关羽“精英杀手”的定位
- 让玩家在高风险点面对守将时有明确强势角色

### 4.2.5 被动技能

#### 固有被动：武圣威压

效果方向：

- 对精英与首领造成额外伤害
- 攻击命中时更容易打出击退或打断效果

#### 觉醒被动：拖刀势

效果方向：

- 连续不受击时，下次重斩伤害提高
- 重斩命中可附带破甲

### 4.2.6 搜打撤专属能力

| 维度 | 设计 |
|---|---|
| 搜 | 清理守点敌人更快，减少高价值点争夺时间 |
| 打 | 对精英、守将、阻拦单位优势极大 |
| 撤 | 撤离时通过高击退能力强行开路，但机动性不如赵云 |

### 4.2.7 局内强化方向

#### 1. 重斩爆发流

- 主动技能增伤
- 重击暴击率提升
- 蓄力缩短

#### 2. 精英斩首流

- 对精英伤害提升
- 破甲层数增加
- 击杀精英后获得短时强化

#### 3. 霸体压制流

- 技能释放期间减伤
- 普攻末段带击退
- 正面硬拼能力提升

### 4.2.8 局外养成映射

| 养成项 | 局内体现 |
|---|---|
| 等级 | 基础伤害与生命更明显增长 |
| 技能等级 | 斩击伤害、破甲值、冷却优化 |
| 星级 | 解锁精英斩杀强化 |
| 觉醒 | 提升重斩威胁与打断能力 |
| 专属强化 | 扩大关羽对精英战的统治力 |

### 4.2.9 首版实现优先级

关羽属于 **P0.5**，必须进入首发，但实现复杂度略高于赵云。

原因：

- 负责建立“重斩猛将”的强感知角色
- 负责首版精英战体验上限

## 4.3 黄忠

### 4.3.1 基础信息

| 项目 | 内容 |
|---|---|
| 名称 | 黄忠 |
| 阵营 | 蜀 |
| 职业 | 射手 |
| 武器 | 弓 |
| 获取方式 | 首批解锁武将 |

### 4.3.2 核心定位

黄忠是“远程稳搜型主角”。

他在搜打撤中的职责是：

- 更安全地清理敌人
- 更稳定地处理中远距离威胁
- 更适合保守玩家做稳搜打法

一句话概括：

**黄忠不是最猛的近战角色，但他最适合把风险压低后慢慢赚。**

### 4.3.3 普攻模组

设计方向：稳定、精准、远程压制。

| 项目 | 设计 |
|---|---|
| 攻击段数 | 单发 / 双连射 |
| 攻击范围 | 远 |
| 节奏 | 中 |
| 动作感 | 拉弓、放箭、命中反馈清晰 |
| 目标手感 | 站位好时非常舒服，站位差时压力明显 |

### 4.3.4 主动技能

#### 固有主动技：百步穿杨

效果定义：

- 锁定直线前方或优先锁定高威胁目标
- 打出一支高伤穿透箭
- 对远距离目标有额外暴击或伤害加成

建议参数方向：

- 冷却：9 秒
- 穿透：可穿透 2-4 名普通敌人
- 对精英带有弱点打击效果

### 4.3.5 被动技能

#### 固有被动：老当益壮

效果方向：

- 与目标距离越远，伤害越高
- 连续远程命中可叠加精准层数

#### 觉醒被动：神射

效果方向：

- 一定概率直接标记高价值敌人或弱点目标
- 对标记目标的伤害显著提升

### 4.3.6 搜打撤专属能力

| 维度 | 设计 |
|---|---|
| 搜 | 更容易在远处发现高价值点或危险敌人 |
| 打 | 远程处理弓手、斥候、精英更安全 |
| 撤 | 可在撤离前提前清路，但被近身后更吃操作 |

### 4.3.7 局内强化方向

#### 1. 穿透狙杀流

- 穿透数量增加
- 对直线目标额外伤害
- 对精英弱点伤害提升

#### 2. 稳搜控场流

- 远程减速
- 预警范围增加
- 搜索点附近输出强化

#### 3. 暴击点杀流

- 远距暴击率提升
- 标记目标伤害提升
- 首箭伤害增幅

### 4.3.8 局外养成映射

| 养成项 | 局内体现 |
|---|---|
| 等级 | 基础输出与生存底盘提升 |
| 技能等级 | 穿透箭伤害、命中数、冷却优化 |
| 星级 | 提升对标记目标收益 |
| 觉醒 | 明显强化稳搜与弱点点杀 |
| 专属强化 | 加强安全输出与远距判断能力 |

### 4.3.9 首版实现优先级

黄忠属于 **P0.5**。

原因：

- 提供与近战完全不同的玩法感受
- 负责“稳搜玩家”这一类人群
- 为后续弓将模板打底

## 4.4 诸葛亮

### 4.4.1 基础信息

| 项目 | 内容 |
|---|---|
| 名称 | 诸葛亮 |
| 阵营 | 蜀 |
| 职业 | 奇谋谋士 |
| 武器 | 羽扇 / 阵法 |
| 获取方式 | 中期解锁或首发高级武将 |

### 4.4.2 核心定位

诸葛亮是“控场收益型主角”。

他在搜打撤中的职责是：

- 提升整局资源收益
- 提供范围控场和局势掌控
- 让玩家感受到不同于纯输出角色的策略价值

一句话概括：

**诸葛亮不是单点最暴力的角色，但他最能决定一局值不值得继续贪。**

### 4.4.3 普攻模组

设计方向：范围、术法、轻控。

| 项目 | 设计 |
|---|---|
| 攻击段数 | 连续术法弹 / 小范围风刃 |
| 攻击范围 | 中远 |
| 节奏 | 中慢 |
| 动作感 | 扇动、法阵、投射 |
| 目标手感 | 不追求单发爆炸，追求稳定控场与空间压制 |

### 4.4.4 主动技能

#### 固有主动技：八阵风灯

效果定义：

- 在前方或脚下展开一片阵法区域
- 区域内敌人减速、持续受伤
- 特定强化后可附加沉默、易伤、击退

建议参数方向：

- 冷却：12 秒
- 持续时间：4-6 秒
- 范围：中等

设计意图：

- 体现诸葛亮的“控场”而不是“蛮力”
- 提升搜点和撤离时的局势处理能力

### 4.4.5 被动技能

#### 固有被动：观势

效果方向：

- 更容易识别高价值搜索点
- 更容易看到精英或危险区域预警

#### 觉醒被动：借东风

效果方向：

- 控场区域内敌人受到额外增伤
- 事件收益、稀有资源概率、撤离条件触发效率提高

### 4.4.6 搜打撤专属能力

| 维度 | 设计 |
|---|---|
| 搜 | 高价值点识别能力最强，事件成功率更高 |
| 打 | 控场优秀，能压低战斗风险 |
| 撤 | 撤离点读条时可铺设控场区域，缓解追兵压力 |

### 4.4.7 局内强化方向

#### 1. 阵法控场流

- 阵法范围扩大
- 阵法持续时间增加
- 敌人减速更强

#### 2. 收益扩张流

- 高价值点刷新提升
- 稀有资源出现率提升
- 撤离结算额外收益

#### 3. 风术增伤流

- 阵法内敌人易伤
- 技能伤害提升
- 精英在阵中受额外伤害

### 4.4.8 局外养成映射

| 养成项 | 局内体现 |
|---|---|
| 等级 | 术法伤害与生存底盘提升 |
| 技能等级 | 阵法范围、持续时间、减速幅度提升 |
| 星级 | 强化事件与收益向能力 |
| 觉醒 | 大幅增强“搜”和“撤”的控制价值 |
| 专属强化 | 让诸葛亮成为高收益地图的优选角色 |

### 4.4.9 首版实现优先级

诸葛亮属于 **P1**。

原因：

- 差异化极强，但实现复杂度比前三人高
- 适合作为首版中后期解锁武将
- 可以在总纲成立后增强产品深度

## 六、局外养成如何映射到局内

### 6.1 武将等级

作用：

- 提升基础攻击
- 提升基础生命
- 提升少量防御与机动底盘

原则：

- 数值成长存在，但不喧宾夺主
- 武将差异主要仍靠技能和特性

### 6.2 技能升级

作用：

- 提升主动技能伤害
- 降低冷却
- 强化附带效果

这是最重要的直观成长来源之一。

### 6.3 星级/升阶

作用：

- 解锁技能附加机制
- 打开新的角色流派
- 增强角色核心标签

举例：

- 赵云 3 星后突进可追加一次穿刺
- 关羽 3 星后重斩附带破甲
- 黄忠 3 星后远距命中可叠精准
- 诸葛亮 3 星后阵法可附带易伤

### 6.4 觉醒

作用：

- 改变角色上限
- 明确角色终局定位

觉醒应该是“风格放大器”，而不是单纯数值放大器。

## 七、首版局内强化池建议

为避免每个武将都要单独做大量强化词条，首版采用：

**通用强化池 + 武将专属强化池**

### 7.1 通用强化池

所有武将都可能获得：

- 攻击提升
- 暴击提升
- 减伤提升
- 搜索速度提升
- 撤离读条减伤
- 战利品概率提升

### 7.2 武将专属强化池

每名武将额外拥有 6-10 条偏专属强化。

例如：

#### 赵云专属

- 七进七出次数 +1
- 突进后获得护盾
- 穿刺命中回血

#### 关羽专属

- 青龙断岳范围扩大
- 对精英增伤
- 重斩打断强化

#### 黄忠专属

- 百步穿杨穿透数提升
- 远距暴击增强
- 标记目标追加伤害

#### 诸葛亮专属

- 阵法持续时间延长
- 阵法附带易伤
- 搜索点稀有率提升

## 九、当前结论

首版武将系统必须做到两件事：

1. **玩家一上手就能感知不同武将真的不一样**
2. **玩家切换武将时不会因为重练成本太高而放弃尝试**

因此首版最佳策略是：

- 数量少
- 差异清晰
- 共享成长高
- 技能识别强

这四点必须同时成立。
